Корак 9.
---------------------

У ћелију B17 упиши следећу формулу:
::

    = MIN(B2:B16)


Ова формула ће израчунати најмањи (MIN) од бројева B2, B3, ..., B16 (B2:B16) и ту вредност уписати у ћелију B17:
